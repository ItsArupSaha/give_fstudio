# **App Name**: Gaura-vāṇī Institute for Vaiṣṇava Education (GIVE)

## Core Features:

- Home Page Hero: Displays the mission statement and prominent call-to-action buttons.
- Courses Catalog: Displays available courses with registration details.
- About Section: Presents the institute's mission, vision, and history.
- Student Testimonials: Shows student feedback and testimonials.
- FAQ Section: Provides an accordion-style FAQ for quick answers.
- Multimedia Resources: Links to external videos, audio, and resource pages.
- Contact Information: Displays a contact form and institute details like address, phone number, email, and social links.

## Style Guidelines:

- Primary color: Light, desaturated green (#C8E6C9) to evoke a sense of calm and growth.
- Background color: Very light desaturated green (#F1F8E9).
- Accent color: A slightly more saturated green (#A5D6A7) to highlight interactive elements and call-to-actions.
- Font pairing: 'Belleza' for headlines (sans-serif), and 'Alegreya' (serif) for body text to balance modernity with readability.
- Mobile-first design approach using a responsive grid system for adaptability across various screen sizes.
- Subtle fade-in effects on scroll to add dynamism to the website.
- Hover effects on cards and buttons to provide visual feedback and enhance interactivity.
